# QR Code Generator

QR Code Generator is a simple web application that allows users to generate QR codes from text or URLs. It utilizes HTML, CSS, JavaScript, and the qrcode.js library to generate QR codes on the client side.

## Features

- Generate QR codes from text or URLs.
- Customize QR code size, colors, padding, and frame design.
- Instantly view and download generated QR codes.
- Responsive design for use on desktop and mobile devices.

## Usage

1. Clone the repository or download the ZIP file.
2. Open the `index.html` file in a web browser.
3. Enter the text or URL you want to encode into a QR code.
4. Click the "Generate QR Code" button.
5. Your QR code will be displayed below the input field.
6. Optionally, adjust the QR code options such as size, colors, padding, and frame design.
7. Click the "Download QR Code" button to save the QR code image.

## Customization

You can customize the following aspects of the generated QR codes:

- **Size**: Adjust the width and height of the QR code.
- **Colors**: Choose the color of the QR code's dark and light modules.
- **Padding**: Add padding around the QR code for better readability.
- **Frame Design**: Customize the design of the QR code's frame.
